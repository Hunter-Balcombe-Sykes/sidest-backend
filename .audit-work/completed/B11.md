# B11 — R2 orphan cleanup on variant failure

## Summary

Added `cleanupR2Artifacts()` to both image and video variant jobs. Called from `failed()` so any terminal failure (exhausted retries) triggers a best-effort R2 delete of the original and partial variants. Errors during cleanup are logged as warnings and swallowed — storage hygiene should never mask the original processing failure.

## Files Changed

- `app/Jobs/ProcessImageVariantsJob.php` — added `cleanupR2Artifacts()` / wired into `failed()`
- `app/Jobs/ProcessVideoVariantsJob.php` — same pattern, calls `VideoVariantService::deleteVariants()`
